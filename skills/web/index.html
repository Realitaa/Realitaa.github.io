Pengembangan Web

Pengembangan web adalah salah satu skill yang dominan pada saya karena dapat dibagun dengan cepat dan dapat dipilih tingkat kesulitannya. Selain itu, kebanyakan tugas sekolah menggunakan web sebagai penilaian berbasis praktek yang membuat saya lebih terbiasa dengan pengembangan web.

  • Hanya menuliskan kode HTML dan langsung bisa dijalankan di browser.
  • Tidak membutuhkan komputer spesifikasi tinggi.

Bahasa Yang Sering Digunakan

  • HTML
  • PHP

Bootstrap 5

Bootstrap 5 adalah framework komponen CSS yang dapat digunakan dengan mudah dan memiliki banyak komponen. Website menjadi responsif dan menarik. Bootstrap 5 juga memiliki dokumentasi yang lengkap dan mudah dipahami.
